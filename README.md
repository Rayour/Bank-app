# Bank-app

## Описание:

Курсовой проект. 
Проект представляет из себя бекенд часть приложения для работы над банковскими операциями клиентов.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Rayour/Bank-app.git
```
2. Установите зависимости:
```
poetry instal 
```
